 Library Book Management System – Frontend

 Project Description

The **Library Book Management System (Frontend)** is a web application developed using **React.js**.
It provides a simple and user-friendly interface to manage and view library book information.
The frontend application is designed to be responsive and is deployed using **Netlify**.

 Technologies Used

* Frontend Framework: React.js
* Languages: HTML, CSS, JavaScript
* Package Manager: npm
* Version Control: Git & GitHub
* Deployment Platform: Netlify

 Frontend File Structure
Frontend/
│
├── App.jsx
├── index.html
├── package.json
├── package-lock.json
└── react.svg
 File Description
 App.jsx

* Main React component of the application
* Displays the title **Library Book Management System**
* Acts as the root component for rendering UI

index.html

* Main HTML file
* Contains `<div id="root"></div>`
* React app is rendered inside the root element

package.json

* Contains project metadata
* Manages dependencies like React and ReactDOM
* Includes scripts to run and build the project

package-lock.json

* Auto-generated file
* Locks dependency versions
* Ensures consistency across systems

react.svg

* React logo image
* Used for UI branding or icon

 How to Run the Project

1. Install dependencies
npm install

2. Start the development server
npm run dev

4. Open browser and visit
http://localhost:5173

 Deployment

* Project is pushed to GitHub
* Frontend is deployed using Netlify
* Netlify builds and hosts the application automatically
* A public URL is generated

 Conclusion

The frontend of the **Library Book Management System** was successfully developed using React.js and deployed using Netlify. The application is accessible through a public URL and works across devices.
